# /apply-find

> 익숙하지 않은 플레이스홀더가 보이거나 연결된 도구를 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 참조하세요.

정부, 민간, 지자체, 대기업, 해외까지 전방위로 지원사업 공고를 탐색하고 적합도 순으로 정렬합니다.

## 사용법

```
/apply-find                          # 전체 소싱
/apply-find AI·SaaS                  # 키워드 지정
/apply-find --deadline 2026-04       # 마감일 필터
/apply-find AI·SaaS --deadline 2026-03
```

---

## 작동 방식

```
┌─────────────────────────────────────────────────────────────────┐
│                     APPLY FIND                                   │
├─────────────────────────────────────────────────────────────────┤
│  단독 사용 (항상 작동)                                            │
│  ✓ program-sourcer 에이전트가 전방위 웹 검색                      │
│    - 정부 포털 (기업마당, K-Startup, TIPS, 중기부)               │
│    - 민간 AC/VC 프로그램                                          │
│    - 지자체 (서울산업진흥원, 테크노파크, 창업지원센터, 경기기업비서) │
│    - 대기업 오픈이노베이션                                        │
│    - 해외 (YC, Techstars, KOTRA 글로벌)                         │
│  ✓ 지식베이스와 대조 → 적합도 점수 자동 산출                      │
│  ✓ 결과를 적합도 순으로 정렬 출력                                  │
├─────────────────────────────────────────────────────────────────┤
│  강화 모드 (도구 연결 시)                                         │
│  + ~~data enrichment: 추가 기업 데이터 보강                      │
└─────────────────────────────────────────────────────────────────┘
```

## 워크플로우

```
/apply-find AI·SaaS
        │
        ▼
program-sourcer 에이전트 (병렬 검색)
├── 기업마당: "AI SaaS 지원사업 공고 2026"
├── K-Startup: "AI 스타트업 모집 2026"
├── TIPS: "TIPS 2026 모집"
├── 지자체: "서울 AI 스타트업 지원 2026"
├── 민간 AC: "AI 스타트업 액셀러레이팅 모집"
├── 오픈이노베이션: "대기업 AI 스타트업 협업 모집"
└── 해외: "global AI startup program 2026"
        │
        ▼
공고 표준화 → 지식베이스와 대조 → 적합도 산출
        │
        ▼
결과 출력 (적합도 순)
```

## 출력 형식

```markdown
# 지원사업 소싱 리포트 | [날짜]

검색 조건: [키워드] | [마감 조건]
총 X개 공고 발견 (높은 적합도 N개, 검토 필요 M개)

---

## 🟢 높은 적합도 (80%+)

| 순위 | 공고명 | 주관기관 | 마감일 | 적합도 | 지원 규모 |
|-----|-------|---------|-------|-------|----------|
| 1 | 창업성장기술개발 | 중기부 | 03-15 | 92% | 최대 3억 |
| 2 | AI 바우처 | 과기부 | 03-20 | 88% | 최대 3억 |
| 3 | TIPS | 중기부 | 수시 | 85% | 최대 5억 |

## 🟡 검토 필요 (60~79%)

| 순위 | 공고명 | 주관기관 | 마감일 | 적합도 | 비고 |
|-----|-------|---------|-------|-------|------|
| 4 | 서울형 R&D | SBA | 04-01 | 72% | 서울 소재 확인 필요 |
| 5 | 글로벌 액셀러레이팅 | KOTRA | 04-10 | 65% | 해외 진출 계획 필요 |

## ⏰ 마감 임박 (7일 이내)

| 공고명 | 마감일 | D-day | 적합도 | 액션 |
|-------|-------|-------|-------|------|
| 창업성장기술개발 | 03-15 | D-5 | 92% | `/apply-check 창업성장기술개발` |

---

*상세 적합도 분석: `/apply-check [공고명]`*
*바로 작성 시작: `/apply-write [공고명]`*
```

---

## 관련 커맨드

- `/apply-check` — 특정 공고 상세 적합도 분석
- `/apply-write` — 사업계획서 작성 시작
- `/apply-daily` — 전체 진행 현황

## 관련 스킬

이 커맨드 실행 시 `gov-program-knowledge` 스킬이 자동 활성화됩니다.

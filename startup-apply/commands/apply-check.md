# /apply-check

> 익숙하지 않은 플레이스홀더가 보이거나 연결된 도구를 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 참조하세요.

특정 지원사업 공고의 자격 요건을 지식베이스와 대조하여 상세 적합도를 분석합니다. 지원 가능 여부 확인부터 작성 전략까지 한 번에 제공합니다.

## 사용법

```
/apply-check TIPS
/apply-check 창업성장기술개발
/apply-check "예비창업패키지 2026"
```

---

## 작동 방식

```
┌─────────────────────────────────────────────────────────────────┐
│                    APPLY CHECK                                   │
├─────────────────────────────────────────────────────────────────┤
│  단독 사용 (항상 작동)                                            │
│  ✓ 공고 요강 자동 검색 (배점, 자격 요건, 필수 항목)               │
│  ✓ 지식베이스 6개 축 대조 분석                                    │
│  ✓ 충족/보완필요/미충족 분류                                      │
│  ✓ 작성 전략 및 강조 포인트 제안                                   │
│  ✓ 준비 체크리스트 생성                                           │
└─────────────────────────────────────────────────────────────────┘
```

---

## 워크플로우

```
/apply-check TIPS
        │
        ▼
suitability-analyzer 에이전트
├── 웹 검색: TIPS 공고 요강, 평가 기준, 배점
├── KB 로드: company-profile, product, market,
│            financials, track-record, business-model
└── 6개 축 대조 분석
        │
        ▼
결과 출력 + 액션 플랜
```

## 출력 형식

```markdown
# 적합도 분석 | TIPS 2026

## 종합 적합도: 85% 🟢

### ✅ 충족 항목

| 항목 | 요건 | 우리 회사 | 판정 |
|------|------|----------|------|
| 업력 | 7년 이내 | 3년 | ✅ |
| 기술 보유 | 특허/IP | 특허 2건 | ✅ |
| 팀 구성 | 기술 인력 2인+ | 개발자 4명 | ✅ |

### ⚠️ 보완 필요

| 항목 | 요건 | 현재 상태 | 보완 방법 |
|------|------|----------|---------|
| 시장 데이터 | 최신 TAM 출처 | 2024년 자료 | `/kb-update market` |

### 💡 작성 전략

TIPS는 기술 혁신성(40%) 배점이 가장 높습니다.
- 특허 2건 + 기술 차별성을 1순위로 강조
- 추천 운영사: [업종 맞는 운영사 3곳]
- 강조할 표현: "세계 최초/국내 유일/기존 대비 X배"

### 📋 준비 체크리스트

1. [ ] 시장 데이터 최신화 → `/kb-update market`
2. [ ] 사업계획서 작성 시작 → `/apply-write TIPS 2026`
```

## 관련 커맨드

- `/apply-find` — 다른 공고 소싱
- `/apply-write` — 바로 사업계획서 작성
- `/kb-update` — 보완 필요 항목 갱신

## 관련 스킬

이 커맨드 실행 시 다음 스킬이 자동 활성화됩니다:
- `gov-program-knowledge` — 공고 유형별 특성
- `kb-structure` — KB 스키마 참조

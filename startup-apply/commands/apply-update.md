# /apply-update

> 익숙하지 않은 플레이스홀더가 보이거나 연결된 도구를 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 참조하세요.

한 번 쓴 사업계획서를 다른 공고에 재활용합니다. 변경이 필요한 부분만 AI가 식별하고 새로 작성하여 중복 작업을 최소화합니다.

## 사용법

```
/apply-update                                          # 대화형
/apply-update 기존계획서.hwp → 창업성장기술개발 2026
/apply-update 최근작성본.md → AI바우처
/apply-update TIPS_2025.md → 예비창업패키지
```

---

## 작동 방식

```
┌─────────────────────────────────────────────────────────────────┐
│                    APPLY UPDATE                                  │
├─────────────────────────────────────────────────────────────────┤
│  단독 사용 (항상 작동)                                            │
│  ✓ 기존 사업계획서 파싱 → 섹션별 분리                             │
│  ✓ 새 공고 요강과 비교 (추가/수정/재활용 분류)                    │
│  ✓ 변경 부분만 새로 작성 (재활용 섹션은 그대로)                   │
│  ✓ 지식베이스 최신 데이터 자동 반영                               │
│  ✓ 변환 diff 리포트 + 새 Markdown 사업계획서 출력                 │
├─────────────────────────────────────────────────────────────────┤
│  강화 모드 (도구 연결 시)                                         │
│  + ~~knowledge base: 최신 KB 데이터 자동 반영                    │
│  + ~~docs: 원본 파일 자동 로드                                    │
└─────────────────────────────────────────────────────────────────┘
```

## 변환 프로세스

```
/apply-update TIPS_2025.md → 창업성장기술개발 2026
        │
        ▼
1. 기존 문서 파싱 → 섹션별 분리
   (기업개요, 기술개발, 시장분석, 재무계획 등)
        │
        ▼
2. 새 공고 요강 분석 (웹 검색)
   - 새 공고의 필수 항목과 배점 파악
   - 기존 문서 섹션과 비교
        │
        ▼
3. 섹션별 분류

   ♻️  재활용 가능: 기업개요, 팀 구성
       → 지식베이스 최신 데이터만 반영

   ✏️  수정 필요: 기술 개발 내용
       → 새 공고 기준에 맞게 재작성

   ➕  추가 필요: 수행 실적 (기존 공고에 없던 섹션)
       → 새로 작성

   🗑️  삭제: [기존 공고 전용 항목]
        │
        ▼
4. 변경분 작성 (bizplan-writer 에이전트)
        │
        ▼
5. 최종 조립 → 새 Markdown 사업계획서
```

## 출력 예시

```markdown
# 변환 완료: TIPS 2025 → 창업성장기술개발 2026

## 변환 요약

| 분류 | 섹션 수 | 내용 |
|------|--------|------|
| ♻️ 재활용 | 5개 | 기업개요, 팀 구성, 시장분석, BM, 기대효과 |
| ✏️ 수정 | 3개 | 기술 개발 내용, 재무 계획, 자금 사용 |
| ➕ 추가 | 2개 | 수행 실적, 추진 일정 |
| 🗑️ 삭제 | 1개 | TIPS 운영사 관련 내용 |

## 변경된 내용

### 기술 개발 내용 (수정)
- TIPS: 기술 혁신성/독창성 중심 → 창업성장기술개발: 연차별 개발 목표 + 성능 지표 추가

### 재무 계획 (수정)
- KB 최신 데이터 반영: MRR 2,000 → 2,800만원

### 수행 실적 (신규)
- track-record KB에서 자동 추출하여 작성

---
→ 전체 사업계획서: [파일명]_창업성장기술개발_2026.md
→ HWPX 내보내기: `/apply-export 창업성장기술개발`
```

---

## 관련 커맨드

- `/apply-write` — 처음부터 새로 작성
- `/apply-export` — HWPX 파일로 내보내기
- `/kb-update` — 변환 전 KB 최신화

## 관련 스킬

이 커맨드 실행 시 `bizplan-writing` 스킬이 자동 활성화됩니다.

# /market-opportunity

> 익숙하지 않은 플레이스홀더가 보이거나 연결된 도구를 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 참조하세요.

TAM (Total Addressable Market), SAM (Serviceable Available Market), SOM (Serviceable Obtainable Market) 시장 규모 분석을 수행합니다. Top-down, Bottom-up, Value Theory 3가지 방법론으로 교차 검증하여 투자자 프레젠테이션용 시장 기회 슬라이드를 생성합니다.

## 사용법

```bash
/market-opportunity                         # 대화형으로 정보 수집
/market-opportunity "한국 B2B SaaS CRM"     # 시장 지정
/market-opportunity "중소기업 회계 자동화"   # 구체적 세그먼트
```

## 필요한 정보

### 1단계: 제품/시장 정의

**필수 입력**:
- 제품/서비스 설명 (한 줄)
- 타겟 고객 (ICP)
  - B2B: 산업, 직원 수, 매출 규모, 지역
  - B2C: 연령, 지역, 관심사, 소득 수준
- 가격 모델
  - SaaS: ARPA (월/년)
  - 마켓플레이스: Take Rate (%)
  - 컨슈머: ARPU
  - B2B: ACV
- 지역 (한국, 미국, 글로벌)

**선택 입력**:
- 주요 경쟁사 (있으면)
- 현재 트랙션 (베타 고객 ARPA 등)

# 시장 규모 분석 상세 계산

## Top-down 방법론

### 1단계: 글로벌 시장 크기
- 소스: [Gartner "CRM Software Market, Worldwide, 2024"]
- 글로벌 CRM 시장: $69B (2024) → $96B (2028, 8.5% CAGR)

### 2단계: 지역 필터 (한국)
- 한국 GDP 대비 IT 지출: 2% (World Bank)
- 한국 CRM = $69B × 2% = **$1.38B**

**검증**:
- KAIT "한국 IT 서비스 시장": $50B
- CRM은 IT 서비스의 2.5%
- $50B × 2.5% = $1.25B (±10% 일치 ✓)

### 3단계: 세그먼트 필터
- 중소기업 (직원 10-500명): 65% (중소벤처기업부)
- SMB CRM = $1.38B × 65% = **$897M**

### 4단계: 제품/채널 필터
- 클라우드 기반: 45% (온프레미스/Excel 제외)
- Cloud SMB CRM = $897M × 45% = **$403M**

### 5단계: 채택률
- 클라우드 전환 의향: 50% (베타 고객 인터뷰)
- SAM = $403M × 50% = **$201.5M ≈ $200M**

### 6단계: SOM (5년 점유율)
- 파편화 시장 (Top 3 각 15%)
- 5년 목표: 5% 점유율
- SOM = $200M × 5% = **$10M**

**TAM/SAM/SOM 요약**:
- TAM: $1.38B (한국 CRM 전체)
- SAM: $200M (중소 클라우드 CRM)
- SOM: $10M (5년 목표)

## Bottom-up 방법론

### 1단계: 타겟 고객 수
- 한국 중소기업 (직원 10-500명): 150,000개 (통계청)
- 현재 CRM 사용: 80% = 120,000개
- 클라우드 전환 의향: 50% = **60,000개 (TAM 고객 수)**

### 2단계: ARPA 설정
- 베타 고객 20개 평균: ₩48만/월 ($400)
- 경쟁사 A (Zoho) 공시: $50/user × 평균 8 users = $400/월
- **ARPA: $400/월 = $4,800/년**

### 3단계: TAM 계산
- TAM = 60,000개 × $4,800 = **$288M/년**

(한국 환율 조정 시: ₩360만/년/고객 × 60,000 = ₩2,160억 ≈ $180M)
→ 보수적 추정: **$180M**

### 4단계: SAM (도달 가능)
- Direct Sales 팀 5명 → 각 100개 커버 = 500개 (0.8%)
- Inbound (SEO/Content) → 10% 도달
- 총 도달률: 10%
- SAM 고객 수 = 60,000 × 10% = 6,000개
- SAM = 6,000 × $4,800 = **$28.8M ≈ $30M**

### 5단계: 전환율 (SOM)
- Lead → Opportunity: 15%
- Opportunity → Customer: 25%
- Overall: 3.75%
- 5년 누적 고객 = 6,000 × 3.75% × 5 (연도별 누적) = 약 200개
- SOM = 200 × $4,800 = **$960K ≈ $1M (첫 해)**
- 5년 복리 성장 (연 100% → 50% → 30% → 20% → 15%):
  - Y1: $1M
  - Y2: $2M
  - Y3: $3.6M
  - Y4: $5.4M
  - Y5: **$6.2M**

**TAM/SAM/SOM 요약**:
- TAM: $180M (60K 고객)
- SAM: $30M (6K 도달 가능)
- SOM: $6.2M (5년 200개 고객)

## Value Theory 방법론

### 1단계: 현재 솔루션 비용
**중소기업 평균 CRM 관련 비용** (연간):
- 레거시 CRM 라이선스: ₩1,200만 (사용자 10명)
- IT 관리 인건비: ₩600만 (유지보수)
- 교육 비용: ₩200만

또는 **수작업 비용**:
- 영업 담당자 시간 낭비: 주 5시간 × 4주 × ₩7만/시간 = ₩140만/월
- 기회 손실 (놓친 Deal): 월 1건 × ₩500만 = ₩500만/월
- **총 비용: ₩640만/월 = ₩7,680만/년 ($64K)**

**보수적 추정**: ₩2,000만/년 채택

### 2단계: 우리 솔루션 가치
- 우리 가격: ₩600만/년 ($5K)
- 절감액: ₩2,000만 - ₩600만 = **₩1,400만/년 ($11.7K)**
- ROI: 233%

### 3단계: 지불 의향
- 일반 원칙: 절감액의 20-30% 지불 의향
- ₩1,400만 × 25% = ₩350만
- 우리 가격 ₩600만은 절감액의 43% → **적정 범위** ✓

### 4단계: TAM 계산
- 타겟 고객: 60,000개 (레거시 CRM 교체 대상)
- TAM = 60,000 × ₩600만 = **₩3,600억 ($300M)**

### 5단계: SOM (5년 채택률)
- 신규 카테고리 아님 (기존 대체) → 채택률 10-20%
- 5년 목표: 10% = 6,000개
- SOM = 6,000 × ₩600만 = **₩360억 ($30M)**

**TAM/SAM/SOM 요약**:
- TAM: $300M (60K 고객 × 지불 의향)
- SAM: $300M (대체재 시장 = TAM)
- SOM: $30M (5년 10% 채택)

## 팁

- **3가지 모두 실행**: 1가지만 하면 검증 불가. 최소 2가지
- **데이터 소스 명시**: "Gartner 2024", "통계청 2023" 등 출처 필수
- **가정 문서화**: "중소기업 = 10-100명"처럼 정의 명확히
- **보수적 선택**: 투자자는 항상 "과대 추정" 의심. 하향 조정 유리
- **역산 검증**: SOM에서 필요 고객 수 역산 → 실행 가능한지 체크
  - 예: $5M SOM ÷ $50K ACV = 100개 고객. 5년 내 달성 가능?
- **성장 시장 강조**: 정체 시장(<5%)보다 10%+ 성장 시장이 유리
- **한국 조정**: 미국 대비 시장 20-30% 작고, ACV 30-50% 낮음 고려
- **분기별 업데이트**: 실제 데이터로 가정 검증 및 수정
- **민감도 분석**: ARPA ±20%, 전환율 ±2%p 시나리오 제시
- **시각화**: TAM/SAM/SOM 깔대기 다이어그램, 성장 차트

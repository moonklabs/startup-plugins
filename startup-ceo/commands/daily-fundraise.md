# /daily-fundraise

> 익숙하지 않은 플레이스홀더가 보이거나 연결된 도구를 확인하려면 [CONNECTORS.md](../CONNECTORS.md)를 참조하세요.

오늘 가장 중요한 펀드레이징 활동을 명확하게 파악합니다. 이 커맨드는 사용자가 제공하는 정보로 작동하며, 도구를 연결하면 더 풍부해집니다.

## 사용법

```
/daily-fundraise
```

오늘의 투자자 파이프라인, 미팅, 후속 연락 대상을 알려주세요.

---

## 작동 방식

```
┌─────────────────────────────────────────────────────────────────┐
│                  DAILY FUNDRAISE BRIEFING                        │
├─────────────────────────────────────────────────────────────────┤
│  단독 사용 (항상 작동)                                            │
│  ✓ 사용자 제공: 오늘의 투자자 미팅, 주요 파이프라인, 후속 연락     │
│  ✓ 정리: 우선순위별 하루 실행 계획                                 │
│  ✓ 딜소싱: 새로운 투자자 타겟 제안                                 │
│  ✓ 출력: 2분 만에 읽을 수 있는 스캔 가능한 브리핑                   │
├─────────────────────────────────────────────────────────────────┤
│  강화 모드 (도구 연결 시)                                         │
│  + ~~calendar: 오늘의 투자자 미팅 자동 가져오기                    │
│  + ~~CRM: 투자자 파이프라인, 단계별 현황, 후속 연락 대상            │
│  + ~~email: 투자자의 미읽음 메일, 답장 대기 중인 메일              │
│  + ~~data enrichment: VC 펀드 최신 뉴스, 신규 투자 발표           │
└─────────────────────────────────────────────────────────────────┘
```

## 필요한 정보

**~~calendar 미연결 시:**
> "오늘 어떤 투자자 미팅이 있나요? (캘린더를 붙여넣거나 나열해 주세요)"

**~~CRM 미연결 시:**
> "현재 진행 중인 투자자는? 후속 연락이 필요한 건이 있나요?"

**커넥터가 있는 경우:**
모든 정보를 자동으로 가져와 브리핑만 보여줍니다.

## 우선순위 프레임워크

다음 프레임워크를 사용하여 오늘의 #1 우선순위를 결정합니다:

| 요소 | 가중치 | 확인 사항 |
|------|--------|-----------|
| **마감 임박** | 35% | 텀시트 협상, DD 마감일 |
| **미팅 중요도** | 30% | Lead VC, 파트너 미팅 |
| **파이프라인 리스크** | 20% | 무응답, 정체, 패스 위험 |
| **딜소싱 공백** | 15% | 커버리지 3x 미달 시 |

## CRM 연결 시

- 투자자 파이프라인을 자동으로 가져옵니다

- 무응답·정체 리스크를 플래그합니다
